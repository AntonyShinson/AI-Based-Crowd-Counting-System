
AI-Based Crowd Counting System

This project performs crowd counting using a live camera or uploaded image/video.
The backend runs locally for real-time performance.
The frontend is a clean, professional web UI for method selection.

Run backend:
python backend/run_crowd.py --source 0
python backend/run_crowd.py --source videos/crowd.mp4
python backend/run_crowd.py --source videos/crowd.jpg
